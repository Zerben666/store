项目部署指南
========
## store
### 一、环境准备
1. 导入数据库
推荐使用MySQL数据库进行导入。如果要自行创建用户数据请分别导入sql_src下的sql文件；如果要使用测试数据可以直接导入store.sql
2. 使用idea打开项目后请确认JDK版本为1.8并且你的电脑有JDK1.8环境
3. 重载maven依赖
4. 确认数据库连接池
确认数据库账号密码和MySQL依赖的版本号与你的数据库对应（源项目自带MySQL8的依赖）
5. 启动数据库，启动项目，访问http://localhost:8080/web/index.html

### 二、测试用户说明
> 本项目使用MD5加密密码，所以您不能直接在数据库中看到明文密码。
